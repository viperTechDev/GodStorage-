# GodStorage — Sacred Vault

GodStorage is a privacy-first web app that allows you to hide files inside images.

## Features
- AES-256 encryption
- Steganography (data hidden in pixels)
- Works 100% offline
- No servers, no tracking

## Why?
Because your data should remain yours.

## How it works
Files are compressed, encrypted, and embedded into PNG images.
